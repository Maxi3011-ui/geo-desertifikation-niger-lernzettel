# Geo Desertifikation & Niger Lernzettel

Statische GitHub-Pages-Seite für einen vollständigen Geographie-Lernzettel zu Desertifikation, Niger und nachhaltiger Entwicklung.

## Inhalt

- Dropdown-Auswahl einzelner Themen
- Immer nur ein Thema sichtbar
- Vollständiger Inhalt der Zusammenfassung
- Mobile- und Apple-Watch-freundliche Darstellung

## Deployment

Die Seite läuft direkt über GitHub Pages aus dem Root der `main`-Branch.
